йғЁзҪІй…ҚзҪ®дёӯеҝғд№ӢApolloжңҚеҠЎ

2020-06-09 | еҲҶеёғејҸжңҚеҠЎ

и®ҝй—®зӯ–з•ҘеҰӮеӣҫпјҢApolloж•ҙдҪ“з”ұдёүдёӘйғЁд»¶PortalгҖҒapollo-configserviceе’Ңapollo-adminservice(еҚ з”Ёз«ҜеҸЈ8070, 8080, 8090)з»„жҲҗпјҡ

1 еҗ„жЁЎеқ—жҰӮиҰҒд»Ӣз»Қ

1.1 Config Service

  • жҸҗдҫӣй…ҚзҪ®иҺ·еҸ–жҺҘеҸЈ
  • жҸҗдҫӣй…ҚзҪ®жӣҙж–°жҺЁйҖҒжҺҘеҸЈпјҲеҹәдәҺHttp long pollingпјү
    • жңҚеҠЎз«ҜдҪҝз”ЁSpring DeferredResultе®һзҺ°ејӮжӯҘеҢ–пјҢд»ҺиҖҢеӨ§еӨ§еўһеҠ й•ҝиҝһжҺҘж•°йҮҸ
    • зӣ®еүҚдҪҝз”Ёзҡ„tomcat embedй»ҳи®Өй…ҚзҪ®жҳҜжңҖеӨҡ10000дёӘиҝһжҺҘпјҲеҸҜд»Ҙи°ғж•ҙпјүпјҢдҪҝз”ЁдәҶ4C8Gзҡ„иҷҡжӢҹжңәе®һжөӢеҸҜд»Ҙж”Ҝж’‘10000дёӘиҝһжҺҘпјҢжүҖд»Ҙж»Ўи¶ійңҖжұӮпјҲдёҖдёӘеә”з”Ёе®һдҫӢеҸӘдјҡеҸ‘иө·дёҖдёӘй•ҝиҝһжҺҘпјүгҖӮ
  • жҺҘеҸЈжңҚеҠЎеҜ№иұЎдёәApolloе®ўжҲ·з«Ҝ

1.2 Admin Service

  • жҸҗдҫӣй…ҚзҪ®з®ЎзҗҶжҺҘеҸЈ
  • жҸҗдҫӣй…ҚзҪ®дҝ®ж”№гҖҒеҸ‘еёғзӯүжҺҘеҸЈ
  • жҺҘеҸЈжңҚеҠЎеҜ№иұЎдёәPortal

1.3 Portal

  • жҸҗдҫӣWebз•Ңйқўдҫӣз”ЁжҲ·з®ЎзҗҶй…ҚзҪ®
  • йҖҡиҝҮMeta ServerиҺ·еҸ–Admin ServiceжңҚеҠЎеҲ—иЎЁпјҲIP+PortпјүпјҢйҖҡиҝҮIP+Portи®ҝй—®жңҚеҠЎ
  • еңЁPortalдҫ§еҒҡload balanceгҖҒй”ҷиҜҜйҮҚиҜ•

1.4 Client

  • ApolloжҸҗдҫӣзҡ„е®ўжҲ·з«ҜзЁӢеәҸпјҢдёәеә”з”ЁжҸҗдҫӣй…ҚзҪ®иҺ·еҸ–гҖҒе®һж—¶жӣҙж–°зӯүеҠҹиғҪ
  • йҖҡиҝҮMeta ServerиҺ·еҸ–Config ServiceжңҚеҠЎеҲ—иЎЁпјҲIP+PortпјүпјҢйҖҡиҝҮIP+Portи®ҝй—®жңҚеҠЎ
  • еңЁClientдҫ§еҒҡload balanceгҖҒй”ҷиҜҜйҮҚиҜ•

2 еҮҶеӨҮйғЁзҪІ

зӮ№еҮ»дёӢиҪҪпјҡapollo-portalгҖҒapollo-configserviceе’Ңapollo-adminserviceгҖӮ

еҮҶеӨҮSQLж–Үд»¶пјҡapolloportaldb.sqlпјҢapolloconfigdb.sqlгҖӮ

иӢҘжҳҜз”ұжәҗз Ғжү“еҢ…ж–Үд»¶пјҢеҲҷclone apolloпјҢдҪҝз”Ёscripts/build.shиҝӣиЎҢжү“еҢ…гҖӮ

2.1 йғЁзҪІ

еҮҶеӨҮDEVгҖҒPROдҝ©зҺҜеўғпјҢжЁЎжӢҹејҖеҸ‘е’Ңз”ҹдә§зҺҜеўғгҖӮ

apollo-configserviceе’Ңapollo-adminserviceеңЁдёӨеҸ°жңәеҷЁдёҠйғҪиҝӣиЎҢйғЁзҪІгҖӮ

PortalйҡҸдҫҝйғЁзҪІеҲ°е“ӘдёҖеҸ°йғҪеҸҜд»ҘгҖӮ

2.1.1 configserviceгҖҒadminserviceйғЁзҪІ

дёӨиҖ…config\application-github.propertiesзӣёеҗҢгҖӮ

жӯӨеӨ„дёҚеұһдәҺй«ҳеҸҜз”ЁйғЁзҪІпјҢеҸӘжҳҜз®ЎзҗҶеӨҡзҺҜеўғпјҢжүҖд»Ҙеҗ„дёӘзҺҜеўғдҪҝз”Ёеҗ„иҮӘзҡ„MySQLеҚіеҸҜгҖӮ

2.1.2Portal

иЎЁи®ҫзҪ®

ApolloPortalDB.ServerConfigдёӯзҡ„apollo.portal.envsпјҲдҫӢеҰӮи®ҫзҪ®дёәпјҡdev,proпјҢйҖ—еҸ·еҲҶеүІпјүгҖӮиҜҘеҸӮж•°з”ЁдәҺжҢҮе®ҡзҺ°еңЁе·Іжңүзҡ„зҺҜеўғгҖӮ

config\apollo-env.propertiesдҝ®ж”№пјҡ

config\application-github.properties

иҮіжӯӨпјҢжүҖжңүдҝ®ж”№еқҮе·Іе®ҢжҲҗгҖӮ

иҝҗиЎҢapollo-configserviceе’Ңapollo-adminserviceзҡ„scripts\startup.shи„ҡжң¬гҖӮ

Portalзӯүеҫ…е…¶д»–жЁЎеқ—еҗҜеҠЁеҲ°дҪҚеҗҺпјҢеҗҜеҠЁгҖӮ

е®ҢжҲҗжңӘжҠҘй”ҷзҡ„жғ…еҶөдёӢпјҢи®ҝй—®Portalең°еқҖпјҡxxxxxxx:8070пјҢиҙҰеҸ·/еҜҶз Ғпјҡapollo/adminгҖӮ

еҗҜеҠЁж—¶пјҢиӢҘйҒҮеҲ°mkdir: cannot create directory вҖҳ/opt/logsвҖҷ: Permission deniedпјҢеҲҷйңҖиҰҒзЎ®и®Өз”ЁжҲ·жҳҜеҗҰжңүжқғйҷҗпјҢжІЎжңүеҲҷдҝ®ж”№scripts/startup.shдёӯзҡ„LOG_DIRгҖӮ

зӮ№еҮ»еҸідёҠи§’вҖңзі»з»ҹдҝЎжҒҜвҖқпјҡ

еҸҜд»ҘзңӢеҲ°йғЁзҪІзҡ„зҺҜеўғиҠӮзӮ№дҝЎжҒҜпјҡ

3 й«ҳеҸҜз”ЁйғЁзҪІ(йӣҶзҫӨдҪҝз”Ё)

еҒҮи®ҫжҲ‘д»¬е°ҶdevзҺҜеўғи®ҫзҪ®дёәй«ҳеҸҜз”ЁгҖӮ

3.1 й…ҚзҪ®ж–Үд»¶дҝ®ж”№

дҝ®ж”№Portalй…ҚзҪ®ж–Үд»¶config\apollo-env.propertiesпјҡ

дҝ®ж”№configserviceгҖҒadminserviceй…ҚзҪ®ж–Үд»¶config\application-github.propertiesпјҢж•°жҚ®еә“еқҮжҢҮеҗ‘еҗҢдёҖж•°жҚ®еә“пјҡ

3.2 ж•°жҚ®еә“й…ҚзҪ®дҝ®ж”№

ApolloConfigDB.ServerConfigж•°жҚ®еә“иЎЁдҝ®ж”№еҸӮж•°eureka.service.urlпјҡ

еҰӮеӣҫжүҖзӨәзҡ„дҝ®ж”№зҡ„ж„ҸжҖқжҳҜпјҡdevзҺҜеўғдёӢAгҖҒBдёӨеҸ°жңәеҷЁпјҢе°Ҷconfig Serverзҡ„дҝЎжҒҜе…ЁйғЁеЎ«е…ҘпјҲиҝҷйҮҢдёӨеҸ°жңәеҷЁжҢҮеҗ‘зҡ„MySQLжҳҜдёҖеҸ°пјүгҖӮеҗҢж ·зҡ„пјҢиҰҒжҳҜеҠ дәҶдёҖдёӘProз”ҹдә§зҺҜеўғпјҢе°ұеңЁProзҺҜеўғзҡ„MySQLдёҠжҠҠconfig Serverзҡ„ең°еқҖпјҢеЎ«е…Ҙeureka.service.urlеҸӮж•°дёӯгҖӮ

д»ҘдёҠдҝ®ж”№е®ҢжҲҗеҗҺпјҢPortalе°ұдјҡиҮӘеҠЁеҲҶеҸ‘иҜ·жұӮдәҶгҖӮ

жӣҙдёәз»ҶиҠӮйғЁеҲҶиҜ·еҸӮйҳ…е®ҳж–№жүӢеҶҢпјҢеҲҶеёғејҸйғЁзҪІжҢҮеҚ—гҖӮ

иҝҷйҮҢзҡ„й…ҚзҪ®жҢҮзҡ„жҳҜпјҢдёҖдёӘзҺҜеўғдёӢпјҢжңүеӨҡдёӘиҠӮзӮ№жҸҗдҫӣжңҚеҠЎпјҢйӮЈиҝҷйҮҢеЎ«е…Ҙзҡ„е°ұжҳҜеҪ“еүҚзҺҜеўғзҡ„eurekaй…ҚзҪ®гҖӮ

жҜҸдёӘзҺҜеўғеҸӘеЎ«е…ҘиҮӘе·ұзҺҜеўғзҡ„eurekaжңҚеҠЎең°еқҖпјҢжҜ”еҰӮFATзҡ„apollo-configserviceжҳҜ1.1.1.1:8080е’Ң2.2.2.2:8080пјҢUATзҡ„apollo-configserviceжҳҜ3.3.3.3:8080е’Ң4.4.4.4:8080пјҢPROзҡ„apollo-configserviceжҳҜ5.5.5.5:8080е’Ң6.6.6.6:8080пјҢйӮЈд№Ҳпјҡ

  1. еңЁFATзҺҜеўғзҡ„ApolloConfigDB.ServerConfigиЎЁдёӯи®ҫзҪ®eureka.service.urlдёәпјҡ
  1. еңЁUATзҺҜеўғзҡ„ApolloConfigDB.ServerConfigиЎЁдёӯи®ҫзҪ®eureka.service.urlдёәпјҡ
